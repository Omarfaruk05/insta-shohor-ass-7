1. How does Javascript work?

Ans: Javascript is a high leavel , scripting, modern, single threaded(single threaded means it can do one work at a time.) and client side language(Client side means it works in browser). When we write JavaScript code this code goes to the browser, the browser sends this code to the JavaScript engine.(Most of the browser has their own javascript engine like google chrome has V8 engine, fire fox has Spider Monkey etc.) The JavaScript engine compiles this code into machine language, then the computer can understand this code and then we see it in the browser.

2. Difference between Local Storage and Session Storage.

Ans:
Local Storage
  • It is the process of storing data in our device storage.
  • It stored data for life time excepting browser resart.
  • It's data capacity is 5MB per app per browser.
Session Storage
  • It is the porcess of storing data in our browser. .
  • It stored data until the browser tab is oppen.
  • It's capacity is limited only system memory.
